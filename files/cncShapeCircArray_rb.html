cncShapeCircArray.rb Up until now most of the circular array functions have been built in to the basic object types so that shapes like cncShapePolygon knew how to produce an array of them selfs centered around an arbitrary point however cncShapeCircArray can produce a circular array of any cncShape that follows the basic cncShape X,Y protocol.
